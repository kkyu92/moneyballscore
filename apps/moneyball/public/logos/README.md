- `LT.svg` — 롯데 자이언츠

- `NC.svg` — NC 다이노스

## 현재 상태

각 PNG는 네이버 스포츠 CDN(`sports-phinf.pstatic.net/team/kbo/default/`)에서
받아온 KBO 공식 엠블럼. 분석·보도 목적 fair use 범위.

## 실제 공식 로고 교체 절차

1. 구단 공식 사이트, KBO 공식, 또는 Wikimedia Commons에서 SVG/PNG 로고 확보
2. SVG 권장 (벡터 스케일링)
3. 동일 파일명으로 덮어쓰기 (`LG.svg` 자리에 LG 트윈스 공식 SVG)
4. `viewBox="0 0 64 64"` 이상 정사각형 권장 (컴포넌트가 자동 스케일)
5. Vercel 자동 재배포 → 즉시 반영

## 저작권 주의

KBO 구단 로고는 각 구단의 저작권. 분석·보도 목적의 fair use 범위 내 사용 권장.
상업적 굿즈·머천다이즈 사용은 별도 라이선스 필요.
