### Rollout step

각 step 간 ≥7일 간격 + Brier delta 측정 ≥3회 → 안정 확인 후 다음 step

1. **Step 1 — 10% rollout**
   - flag `model-version-v20-rollout = 10` 박제
   - 매 predict cron fire 시 `Math.random() * 100 < 10` 시 v2.0 후보 weights 사용
   - 7일 후 Brier 측정 (v1.8 90% cohort vs v2.0 10% cohort 비교)
   - 안정 (Brier delta < 1pp) 시 Step 2 진행 / 회귀 (-2pp+) 시 즉시 rollback (flag=0)

2. **Step 2 — 50% rollout**
   - flag `model-version-v20-rollout = 50` 박제
   - 7일 후 Brier 측정 (50% cohort 비교)
   - 안정 시 Step 3 / 회귀 시 즉시 rollback (flag=0 또는 10)

3. **Step 3 — 100% rollout (v2.0 production ship)**
   - flag `model-version-v20-rollout = 100`
   - v1.8 cohort 폐기 + v2.0 새 baseline 박제
   - `DEFAULT_WEIGHTS` 코드 swap → SHADOW_V20_WEIGHTS frozen 해제 (자유 변경 가능)

### Rollback path
