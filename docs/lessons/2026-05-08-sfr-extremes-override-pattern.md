# Anti-Pattern: 단일 팩터 극단값이 앙상블 신호를 오버라이드

**카테고리**: anti_pattern / ai_agent  
**발견**: cycle 256 (2026-05-08), W19 SSG vs NC 3연전 분석

---

## Problem

KBO 예측 모델에서 `sfr` (수비효율) 팩터가 `0.72 → 0.72 → 1.0` 극단값을 연속으로 기록한 SSG vs NC 3연전에서, `head_to_head=0.33` (NC 상대전적 우세) 신호가 올바른 방향을 가리켰음에도 모델이 3경기 모두 SSG 승리를 예측하며 전패(0/3).

SFR이 0.5 기준으로 0.22~0.50 편차를 가지는 동안, head_to_head는 0.17 편차였으나 실제 결과에선 head_to_head가 정답이었음.

## Solution (후보)

1. **팩터 극단값 soft cap**: SFR > threshold (예: 0.75) 시 실효 가중치를 일부 할인. `sfr_effective = min(sfr, 0.5 + (sfr-0.5)*0.7)` 형태의 감쇠 함수.
2. **반대 신호 팩터 보정**: 극단 팩터가 있을 때 반대 방향을 가리키는 팩터(head_to_head 등)의 가중치를 상대적으로 높이는 soft disagreement penalty.
3. **n=150 도달 후 분석**: 소표본 경고(95% CI ≈ ±15%p)로 현재 단정 불가. n=150 후 SFR 극단값(>0.7) 케이스 별도 분류 후 통계 검정.

## Results (현재까지)

- W19 SSG vs NC: head_to_head=0.33(NC 우세) 신호 3번 모두 맞음, SFR에 오버라이드됨 → 3연패
- 삼성 vs 키움 2연전: head_to_head=0.40 신호 맞음, recent_form/sfr에 오버라이드 → 2연패
- 전체 영향: 14경기 중 5 miss의 4건이 이 패턴

## Reusability

앙상블 예측 모델 일반:
- **범주형 팩터 극단값 경계**: 단일 지표가 >0.75 또는 <0.25 (즉, 확신 50%p 이상 편차)일 때 다른 지표와 disagreement 시 자동 플래그
- **실용 가이드**: 극단값 감쇠 없이 선형 가중합 사용 시 단일 지표가 앙상블을 납치하는 현상 발생 가능 — 특히 특정 상황(구장, 날씨, 시리즈)에서만 극단값이 나타나는 지표에서 주의

## 관련 데이터

- n=76 (v1.5, 2026-05-08 기준)
- 일요일 누적 적중률: 1/8 = 12.5% (별도 패턴)
- 공식 가중치 변경은 n=150 도달 후 예정
